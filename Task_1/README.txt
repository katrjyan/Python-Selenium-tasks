
Task Description  

 
This small Python program scans a web page at a given URL (which is accepted from the command line and some basic validation 
is done to avoid the input of invalid values.) and retrieves the following information:

1. Number of html form elements (with method="get") present in that page 
2. Number of HTML image tags present in that page   

 

I. Python 3.8.1 version has been used for coding the task 
II. Selenium WebDriver has been used for  web page navigation and interaction
III. Python Validators library has been used for validatiing the URL parameter to be accepted from the command line  
IV. The coding program has been designed to be run on a Code Editor and not by a Pytest
